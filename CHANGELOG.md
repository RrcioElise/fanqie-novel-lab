# Changelog

## 0.2.0 - 2026-04-29

### Added

- 发布中心新增发布自动化：本地上传包、门禁、自动化清单和作者有话说。
- 创作约束新增“降低AI味与文本纹理”“有迹可循的意外反转”“钩子库多样化”。
- 大纲和章节计划新增 hook_type、foreshadowing、reversal_logic。
- 章节工坊新增 AI 味检测、伏笔台账和反转审查质检报告。
- 外网介绍页加入 1920×1080 软件截图展示。

### Changed

- 章节审核入口升级为“章节审核与质检”。
- 正文生成提示强化去 AI 味、具体钩子和伏笔回收要求。
- 发布包 Markdown/TXT 增加作者有话说。

## 0.1.0 - 2026-04-28

### Added

- Streamlit 多页面客户端。
- Electron 桌面壳。
- 公开榜单元数据采集与 CSV 导入。

- Claude CLI provider 支持。
- 大纲生成、人审润色、避撞审查。
- 分场景章节生成、章节编辑、章节润色。
- 章节对纲审核。
- 发布中心：作品档案、上传包、发布队列、作家后台助手。
- GitHub 开源文档、Issue 模板、PR 模板、CI。

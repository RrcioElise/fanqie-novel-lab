# FAQ

## 这是写作工具还是爬虫工具？

核心是本地原创写作工作台。采集能力只用于公开元数据趋势观察，真正的创作流程围绕题材、大纲、人审、章节和发布包管理。

## 能接哪些模型？

支持 OpenAI-compatible API、本地 Ollama、LM Studio，以及本机 Claude CLI 桥接。客户端可保存多个 profile 并切换。

## 生成的大纲和章节会上传吗？

不会。默认写入本地 `outputs/`，并被 `.gitignore` 忽略。

## 可以多人协作吗？

当前更偏单机工具。开源后可通过 GitHub Issues/PR 协作代码；作品内容建议另用私有仓库或文档系统管理。

Local AI Fiction Studio

本地 AI 小说创作工作台。

Fanqie Novel Lab 面向中文网文创作流程：公开元数据观察、原创大纲、审稿润色、避撞审查、章节生成、章节对纲审核和发布包管理。

Windows 下载 macOS 下载 查看源码
这个外网页面只介绍软件和提供下载；不提供在线创作服务，不访问你的本地作品、模型密钥或数据库。

为什么是本地？

模型配置、样本库、生成的大纲和章节都保存在用户自己的电脑上。你可以连接 OpenAI-compatible 中转站、本地 Ollama / LM Studio，或 Claude CLI。

Workflow

从题材到章节，再到发布包。

🍅 数据花园

采集或导入公开榜单元数据，做题材趋势观察和避撞对比。

✍️ 大纲画布

设置题材、读者、钩子和约束，生成可审稿的长篇大纲。

🛡️ 原创避撞

用本地公开元数据检查相似钩子、关键词和设定风险。

📖 章节工坊

按章节数和目标字数生成正文，支持人工编辑和 AI 润色。

✅ 章节审核

对照大纲检查跑题、断纲、冲突未兑现和章末钩子弱等问题。

🚀 发布中心

管理作品档案、章节上传包、TXT / Markdown / JSON 导出。

Install

三步启动。

1
安装环境

安装 Python 3.10+。桌面客户端还需要 Node.js LTS。

2
下载发布包

Windows 解压后运行 .\scripts\setup.ps1；macOS 解压后运行 bash scripts/setup.sh。

3
配置模型

进入“项目设置 → 模型”，填入你的中转站、本地模型或 Claude CLI 配置。

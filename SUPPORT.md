# Support

## 提问前先准备

请尽量附上：

- 操作入口：Web UI / Electron / CLI
- 系统和 Python 版本
- 使用的模型 provider 类型：OpenAI-compatible / Ollama / LM Studio / Claude CLI
- 最小复现步骤
- 已脱敏日志或截图

不要公开粘贴 API Key、Cookie、账户信息、未发布正文或本地数据库。

## 推荐渠道

- Bug：使用 GitHub Issues 的 Bug report 模板。
- 功能建议：使用 Feature request 模板。
- 使用问题：使用 Question 模板或 Discussions（如果仓库已开启）。
- 安全问题：按 `SECURITY.md` 说明私下报告。
